Movie Paradise
Overview
Movie Paradise is a passion project of mine, a dynamic front-end web application that provides comprehensive details about movies. Leveraging a movie data API, the app fetches real-time information, ensuring users have access to the latest movie data. The app boasts an intuitive search functionality, allowing users to easily find movies of interest. As they type in the search bar, the results dynamically update, providing a seamless and engaging user experience.
Technologies Used
HTML
CSS
ReactJs
RestAPI
Goals
  • Fetch movie data from an API
  • Search Movies
  • Get Movie Details including their rating
  • Create a webpage for each movie with its info
